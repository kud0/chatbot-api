📱 WHATSAPP INTEGRATION
================================================================================

3 TYPES OF WHATSAPP BUTTONS

1. Floating Button (Bottom-Right)
   Location:    Every page (fixed position)
   Message:     "Hola! Me gustaría información sobre servicios"
   Features:    Pulse animation, tooltip on hover

2. Service Cards ("Reservar" Button)
   Location:    Homepage and services page
   Message:     "Hola! Me gustaría reservar una cita para [Service Name]"
   Features:    Service-specific pre-filled message

3. CTA Sections (Large Buttons)
   Location:    Homepage hero, end sections, services page
   Message:     Customizable per component
   Features:    Dark background, prominent design

LINK FORMAT
  https://wa.me/34912345678?text=Hola!%20Message%20here
  
  Components:
  • 34912345678 = Phone from barbershop.json (no +)
  • text= = Pre-filled message (URL encoded)
